## OpenSource Release

Orionis Framework is undergoing a **gradual OpenSource release**. The framework has been developed, tested, and is currently serving as the technological foundation for various enterprise systems in production.

:::tip[A framework born in production]
Orionis is not an experimental project. It was born in real-world environments and has been validated in enterprise systems before its public release. The gradual release ensures that every component meets the quality standards the ecosystem deserves.
:::

### Release Status

<Steps>

1. **Framework Core** <Badge text="Completed" variant="success" />

   Core features tested and stable. The framework's core is production-ready.

2. **Documentation** <Badge text="In Progress" variant="caution" />

   Under continuous development and improvement. Each module is documented with practical examples and detailed guides.

3. **Components** <Badge text="In Progress" variant="caution" />

   Gradual release of modules and features. Components are published as they pass quality assurance.

4. **Public Testing** <Badge text="Pending" variant="note" />

   Preparing for community testing. We will soon open channels for feedback and contributions.

</Steps>

:::note[A thank you to the community]
This release process requires time and dedication to ensure the quality, organization, and proper documentation of each component. We appreciate the community's patience and support as we complete this process.
:::

### Versioning Policy

Orionis follows the **Semantic Versioning (SemVer)** standard:

<CardGrid>
  <Card title="Major (X.0.0)" icon="warning">

  </Card>

  <Card title="Minor (0.X.0)" icon="add-document">
    New backward-compatible features. Modules and enhancements that extend capabilities without breaking existing code.
  </Card>

  <Card title="Patch (0.0.X)" icon="approve-check">
    Bug fixes and performance improvements. Safe updates that can be applied without risk.
  </Card>
</CardGrid>

<script>
{`
document.addEventListener('DOMContentLoaded', function() {
    async function fetchVersionInfo() {
        const currentVersionEl = document.getElementById('current-version');
        const dynamicVersionEl = document.getElementById('dynamic-version');
        const statusEl = document.getElementById('version-status');
        try {
            currentVersionEl.textContent = 'Loading...';
            currentVersionEl.classList.add('loading');
            const response = await fetch('https://pypi.org/pypi/orionis/json');
            if (!response.ok) {
                throw new Error('HTTP ' + response.status + ': ' + response.statusText);
            }
            const data = await response.json();
            const version = data.info.version;
            const uploadTime = new Date(data.urls[0]?.upload_time || Date.now());
            currentVersionEl.textContent = 'v' + version;
            currentVersionEl.classList.remove('loading');
            dynamicVersionEl.textContent = 'v' + version;
            const timeSinceUpload = Math.floor((Date.now() - uploadTime.getTime()) / (1000 * 60 * 60 * 24));
            statusEl.textContent = 'OpenSource - Gradual Release (updated ' + timeSinceUpload + ' days ago)';
        } catch (error) {
            currentVersionEl.textContent = 'Not available';
            currentVersionEl.classList.remove('loading');
            dynamicVersionEl.textContent = '';
            statusEl.textContent = 'Error loading version info';
            setTimeout(function() {
                fetchVersionInfo();
            }, 5000);
        }
    }
    fetchVersionInfo();
    setInterval(fetchVersionInfo, 300000);
});
`}
</script>
