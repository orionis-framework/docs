## The Story Behind Orionis

Imagine a team of PHP experts, masters of precision and agility, facing a new technological frontier. The challenges — **RPA, AI, Blockchain, Data Analytics, OCR, and Computer Vision** — demanded more than skill; they required new tools. Enter Python: versatile, powerful, and ready to tackle the future.

But the leap to Python wasn't just about learning a new language; it was about adopting a new mindset. Fundamental questions arose:

<Steps>

1. **How do we structure classes naturally?**

   We needed a system that respected the patterns we had already mastered.

2. **How do we manage the lifecycle of a Request?**

   The input, processing, and response flow had to be intuitive and efficient.

3. **How do we implement robust Middleware?**

   The backbone of modern applications had to be flexible and powerful.

</Steps>

The answer was to create something extraordinary: **a framework that feels like home.** Thus, Orionis was born.

:::note[More than a technological shift]
With Orionis, the leap to Python is a journey of empowerment. It's a community where craftsmanship meets science, elevating development to new heights within Python's powerful ecosystem. Develop fast, innovate effortlessly.
:::

## Real Performance Statistics

:::caution[Unmatched Performance]
Orionis Framework delivers unmatched performance and efficiency in Python. Built on **Granian**, the fastest HTTP server in the Python ecosystem, developed in Rust.
:::

### Supported Interfaces

<Tabs>
  <TabItem label="ASGI" icon="laptop">
    **ASGI** *(Asynchronous Server Gateway Interface)*

    Full compatibility with ASGI interfaces, integrating Orionis with major frameworks and modern Python tools. This allows you to use Orionis alongside the existing ecosystem of asynchronous Python applications without any friction.
  </TabItem>

  <TabItem label="RSGI" icon="rocket">
    **RSGI** *(Rust Server Gateway Interface)*

    Harness the power of Granian and Rust with the RSGI interface, achieving ultra-fast response times and maximum efficiency. This native interface lets you squeeze every drop of performance from the server.
  </TabItem>
</Tabs>
