## La historia detrás de Orionis

Imagina un equipo de expertos en PHP, maestros en precisión y agilidad, frente a una nueva frontera tecnológica. Los retos — **RPA, IA, Blockchain, Analítica de Datos, OCR y Visión por Computadora** — exigían más que habilidad; requerían nuevas herramientas. Así llegó Python: versátil, potente y preparado para afrontar el futuro.

Pero el salto a Python no fue solo aprender un nuevo lenguaje; fue adoptar una nueva mentalidad. Surgieron preguntas fundamentales:

<Steps>

1. **¿Cómo estructuramos las clases de forma natural?**

   Necesitábamos un sistema que respetara los patrones que ya dominábamos.

2. **¿Cómo gestionamos el ciclo de vida de una Request?**

   El flujo de entrada, procesamiento y respuesta debía ser intuitivo y eficiente.

3. **¿Cómo implementamos Middleware robusto?**

   La columna vertebral de las aplicaciones modernas tenía que ser flexible y poderosa.

</Steps>

La respuesta fue crear algo extraordinario: **un framework que se sintiera como en casa.** Así nació Orionis.

:::note[Más que un cambio tecnológico]
Con Orionis, el salto a Python es un viaje de empoderamiento. Es una comunidad donde la artesanía se une a la ciencia, elevando el desarrollo a nuevas alturas dentro del poderoso ecosistema de Python. Desarrolla rápido, innova sin esfuerzo.
:::

## Estadísticas de Rendimiento Reales

:::caution[Rendimiento Inigualable]
Orionis Framework ofrece un rendimiento y eficiencia inigualables en Python. Construido sobre **Granian**, el servidor HTTP más rápido del ecosistema Python, desarrollado en Rust.
:::

### Interfaces soportadas

<Tabs>
  <TabItem label="ASGI" icon="laptop">
    **ASGI** *(Asynchronous Server Gateway Interface)*

    Compatibilidad total con interfaces ASGI, integrando Orionis con los principales frameworks y herramientas modernas de Python. Esto permite usar Orionis junto con el ecosistema existente de aplicaciones asíncronas de Python sin fricción alguna.
  </TabItem>

  <TabItem label="RSGI" icon="rocket">
    **RSGI** *(Rust Server Gateway Interface)*

    Aprovecha la potencia de Granian y Rust con la interfaz RSGI, logrando tiempos de respuesta ultra-rápidos y máxima eficiencia. Esta interfaz nativa permite exprimir cada gota de rendimiento del servidor.
  </TabItem>
</Tabs>
