## Liberación OpenSource

Orionis Framework está en un proceso de **liberación gradual OpenSource**. El framework ha sido desarrollado, probado y está siendo utilizado como base tecnológica de diversos sistemas empresariales en producción.

:::tip[Un framework nacido en producción]
Orionis no es un proyecto experimental. Nació en entornos reales y ha sido validado en sistemas empresariales antes de su publicación abierta. La liberación gradual garantiza que cada componente cumple los estándares de calidad que el ecosistema merece.
:::

### Estado del Proceso

<Steps>

1. **Framework Core** <Badge text="Completado" variant="success" />

   Funcionalidades principales probadas y estables. El núcleo del framework está listo para producción.

2. **Documentación** <Badge text="En progreso" variant="caution" />

   En proceso de desarrollo y mejora continua. Cada módulo se documenta con ejemplos prácticos y guías detalladas.

3. **Componentes** <Badge text="En progreso" variant="caution" />

   Liberación gradual de módulos y funcionalidades. Los componentes se publican a medida que superan las pruebas de calidad.

4. **Testing Público** <Badge text="Pendiente" variant="note" />

   Preparación para pruebas de la comunidad. Próximamente abriremos canales para feedback y contribuciones.

</Steps>

:::note[Agradecimiento a la comunidad]
Este proceso de liberación requiere tiempo y dedicación para asegurar la calidad, organización y documentación adecuada de cada componente. Agradecemos la paciencia y el apoyo de la comunidad mientras completamos este proceso.
:::

document.addEventListener('DOMContentLoaded', function() {
    async function fetchVersionInfo() {
        const currentVersionEl = document.getElementById('current-version');
        const dynamicVersionEl = document.getElementById('dynamic-version');
        const statusEl = document.getElementById('version-status');
        try {
            currentVersionEl.textContent = 'Cargando...';
            currentVersionEl.classList.add('loading');
            const response = await fetch('https://pypi.org/pypi/orionis/json');
            if (!response.ok) {
                throw new Error('HTTP ' + response.status + ': ' + response.statusText);
            }
            const data = await response.json();
            const version = data.info.version;
            const uploadTime = new Date(data.urls[0]?.upload_time || Date.now());
            currentVersionEl.textContent = 'v' + version;
            currentVersionEl.classList.remove('loading');
            dynamicVersionEl.textContent = 'v' + version;
            const timeSinceUpload = Math.floor((Date.now() - uploadTime.getTime()) / (1000 * 60 * 60 * 24));
            statusEl.textContent = 'OpenSource - Liberación Gradual (actualizada hace ' + timeSinceUpload + ' días)';
        } catch (error) {
            currentVersionEl.textContent = 'No disponible';
            currentVersionEl.classList.remove('loading');
            dynamicVersionEl.textContent = '';
            statusEl.textContent = 'Error al cargar información de versión';
            setTimeout(function() {
                fetchVersionInfo();
            }, 5000);
        }
    }
    fetchVersionInfo();
    setInterval(fetchVersionInfo, 300000);
});
`}
</script>
