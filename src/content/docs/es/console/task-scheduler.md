## Programador de Tareas

Orionis Framework incluye un programador de tareas integrado que permite ejecutar comandos de consola de forma programada. Esto es útil para tareas recurrentes como limpieza de cache, generación de reportes o cualquier proceso que necesite ejecutarse automáticamente.

Esta inspirado en frameworks probados y ofrece una api completa para definir tareas, configurar horarios y manejar ejecuciones concurrentes, susbcribirse a eventos y más.

Super completo en un solo lugar...
